# WOJAK Image Style Reference

Shared art-direction for every `[WOJAK]` scene in this video. Prepend/enforce these
rules when generating each image so the batch reads as one series — and as a sibling to
the rest of the channel's videos. Fill in the two **per-video** blanks at the bottom.

## The look
- **Dense, painterly comic-book scenes** with real environments and full foreground /
  midground / background depth. NOT characters floating on white.
- **Cinematic wide** framing (~16:9), single strong vanishing point / eyeline.
- **Palette:** desaturated **teal-grey** world + ONE glowing energy accent (this video's
  accent — see below). Warm gold for anything "elite/sacred"; cold teal for the mundane.
- **Consistent line + grade** across all panels; slight scanline/print texture.

## The faces (critical)
- Wojak / soyjak / NPC-wojak / pepe faces **grafted onto otherwise realistically drawn,
  fully-clothed people** doing real things in a real world. The meme is the **face**,
  never the whole body-gag. No "a pepe with little legs doing X."
- Use face register to characterize: blank NPC-wojak = the crowd/masses; soyjak = the
  excitable/tech-bro/objector; serene pepe = the sage/protagonist; classic wojak = the
  lone everyman who *notices*.

## Text (critical)
- **Diegetic only.** Text may appear ON world-objects: product/shelf labels, screens &
  UI, name-tags, signage, carved inscriptions, tickers, wrist-HUDs.
- **NO floating meme-captions, no editorial subtitles, no label-arrows** naming the joke.
  If a line is explaining the image, delete it.
- **One exception:** deliberate **infographic / taxonomy plates** (a labeled progression
  card, a diagram) — there, clean title/label text IS the intent.

## Humor
- Embedded as **background gags**, environmental storytelling, and world-detail — reward
  the viewer who pauses. Never a caption-level punchline. (Think hidden name-tags, absurd
  product labels, one character doing something ridiculous in the crowd.)

## Motion (for the parallax pass)
- Each scene will be split into a magenta-keyed **foreground** plate and an inpainted
  **background** plate, then given a slow camera dolly+truck. Compose with a clear
  near-subject (foreground) against a deeper environment (background) so the parallax
  has something to separate.
