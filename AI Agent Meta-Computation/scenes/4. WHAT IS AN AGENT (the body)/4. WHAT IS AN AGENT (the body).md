## 4. WHAT IS AN AGENT (the body)

So if intelligence is a fluid — what's an agent? | **[CAM]**

Remember the framing. Computation is the gross, dense, physical layer of this world. So an agent is the *grossest computational form the intelligence can pour itself into.* It's a **body.** A dense, running application that the fluid animates — so it can reach out and touch other computational objects: files, terminals, APIs, other programs, and us.
